# Transformations of Trigonometric Functions

## **General Form**

$y = a f(b(x + c)) + d$

where:

- $ a $ = Amplitude (affects vertical stretch/shrink)
- $ b $ = Frequency (affects period: $ \frac{2\pi}{|b|} $ for sin/cos, $ \frac{\pi}{|b|} $ for tan/cot)
- $ c $ = Phase shift (horizontal shift, $ -\frac{c}{b} $)
- $ d $ = Vertical shift (moves graph up/down)

## **Effects of Transformations (order matters)**

| Transformation         | Effect                                                               |
| ---------------------- | -------------------------------------------------------------------- |
| Period Change ($b$)    | Compresses horizontally if ∣b∣>1; stretches horizontally if 0<∣b∣<1. |
| Phase Shift ($c$)      | Shift left if $ c > 0 $, right if $ c < 0 $                          |
| Reflection ($-b$)      | Reflects across the y-axis                                           |
| Amplitude Change ($a$) | Vertical stretch if $ a > 1 $; vertical compression if $ 0 < a < 1 $ |
| Reflection ($-a$)      | Reflects across the x-axis                                           |
| Vertical Shift ($d$)   | Shift up if $ d > 0 $, down if $ d < 0 $                             |

## **Function-Specific Transformations**

| Function                | Original Range      | Key Transformations                                                                                                                                         |
| ----------------------- | ------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Sine & Cosine**       | $[-1, 1]$           | Amplitude controls peak height, Period determines wave’s length, Phase shift moves left/right, Vertical shift raises/lowers, Reflection flips across x-axis |
| **Tangent & Cotangent** | $(-\infty, \infty)$ | No amplitude (infinite range), Period is $ \frac{\pi}{b} $, Asymptotes appear at undefined points, Reflection inverts increasing/decreasing behavior        |
